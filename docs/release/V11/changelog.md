## 更新日志-测试版

为了确保模块的稳定性，Github推送的版本通常会晚几个版本，如需获取最新版本可以前往网盘获取：

链接: [https://yun.139.com/shareweb/#/w/i/2qieog1Xypz11](https://yun.139.com/shareweb/#/w/i/2qieog1Xypz11)

本次更新内容：

- 新增 Android 16 Hyper OS 3 平板端新版居中布局配置，支持16:9和3:2比例，不支持自定义比例

- 更新 KsuWebUI 安装包，安装后将支持导航栏沉浸

- 使用 Ksu 3.0部分新API能力，提高首屏加载速度

- 千问适配全屏

历史更新内容:

- 引入 Hyper OS 3 部分应用布局优化规则

- 新增 小米平板支持开启Hyper OS 3小米超级岛，只要是Hyper OS 3的小米平板都可以在[系统体验增强]处开启，目前暂时不支持超级岛位置自定义调整，后续考虑支持！

- 移除 哔哩哔哩的首页默认分屏功能，避免异常的崩溃问题。
